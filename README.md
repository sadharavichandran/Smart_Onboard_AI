# Run and deploy your AI Studio app

This contains everything you need to run your app locally.

View your app in AI Studio: https://ai.studio/apps/b9315f6f-d29f-41c6-9dd8-5643f80b35fc

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   `npm install`
2. Create a `.env` file in the project root and set `VITE_GEMINI_API_KEY` to your Gemini API key
3. Run the app:
   `npm run dev`
